# TextField

TextField component with flexible props for input, selection, and customization.

#### Example

<Canvas of={TextField.Default} />

## Usage

```tsx
import { TextField } from "sld-ui";

const Example = () => {
  return (
    <TextField
      label="Username"
      value="JohnDoe"
      onChange={(event) => console.log(event.target.value)}
      placeholder="Enter your username"
      helperText="Your unique username"
      variant="outlined"
      size="medium"
    />
  );
};

export default Example;

#### Arguments
- label string - The label displayed above the input field.
- value string | number - The current value of the input field.
- onChange (event: React.ChangeEvent<HTMLInputElement>) => void - A callback function triggered when the input value changes. It receives the change event as an argument.
- placeholder string - Placeholder text displayed when the input is empty.
- type string - Specifies the type of input (e.g., "text", "password", "email"). Default is "text".
- disabled boolean - Disables the input field, preventing user interaction. Default is false.
- helperText string - Additional text displayed below the input field for context or instructions.
- variant string - Specifies the style of the text field. Possible values: "outlined", "filled", "standard". Default is "outlined".
- multiline boolean - Enables multiline input. Default is false.
- rows number - Sets the number of rows for multiline input.
- select boolean - Enables a dropdown menu instead of a standard input field. Default is false.
- options Array<{ label: string; value: string | number }> - An array of options for the dropdown menu. Each option should have a label and value.
- size string - Specifies the size of the input field. Possible values: "small", "medium". Default is "medium".
- margin string - Defines the margin around the input field. Possible values: "none", "dense", "normal". Default is "none".
- width string - Sets the width of the input field. Default is "100%".
- height string - Sets the height of the input field. Optional.
- ...rest object - Additional props passed to the underlying MUI TextField component.
