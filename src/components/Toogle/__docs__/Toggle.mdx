# Toggle

Toggle component with different props.

#### Example

<Canvas of={Toggle.Default} />

## Usage

```tsx
import { Toggle } from "sld-ui";

const Example = () => {
  return (
    <Toggle
      checked={true}
      label="Enable Notifications"
      onChange={(event) => console.log(event.target.checked)}
      helperText="Toggle to enable or disable notifications."
      disabled={false}
    />
  );
};

export default Example;

#### Arguments
- checked boolean - Indicates whether the toggle is in the "on" (checked) or "off" (unchecked) state.
- label string - The label text displayed alongside the toggle switch.
- onChange (event: React.ChangeEvent<HTMLInputElement>) => void - A callback function invoked when the toggle's state changes. It receives the change event as an argument.
- helperText string - Optional text displayed below the toggle for additional context or instructions.
- disabled boolean - Specifies whether the toggle is disabled, preventing interaction.
- color string - Defines the color of the toggle. Default is "primary".
- size string - Sets the size of the toggle. Possible values: "small", "medium", "large". Default is "medium".
- ...rest object - Additional props passed to the underlying MUI Switch component.
